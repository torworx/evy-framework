Object pooling API implementations.

{@link evyframework.pool.impl.SoftReferenceObjectPool SoftReferenceObjectPool} provides a {@link java.lang.ref.SoftReference SoftReference} based {@link evyframework.pool.ObjectPool ObjectPool}.
